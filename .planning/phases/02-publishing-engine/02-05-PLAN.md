1. KeyboardInterrupt caught
2. Set _interrupted flag (stops new publish tasks)
3. Wait for ongoing publish tasks to complete (with timeout)
4. Write partial artifacts (run.json with completed count)
5. Call worker_pool.cleanup() for graceful DISCONNECT
6. Exit
