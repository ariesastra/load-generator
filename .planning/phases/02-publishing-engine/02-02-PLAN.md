# From STACK.md - asyncio.TaskGroup pattern:
```python
import asyncio

async with asyncio.TaskGroup() as tg:
    for i in range(worker_count):
        tg.create_task(publish_worker(i))
```

# User decisions from CONTEXT.md:
- Pre-connect all workers at startup before publishing begins
- Fail-fast if any worker fails to connect during startup
- Fresh connections per benchmark run
- Default MQTT keepalive (60 seconds)
