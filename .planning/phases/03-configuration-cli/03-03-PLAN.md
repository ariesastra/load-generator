# Configuration schema (from 03-01-PLAN.md)
Required structure for scenario YAML:
```yaml
name: str
description: Optional[str]
message_count: int
worker_count: int
rate_limit: int
qos: int (0, 1, or 2)
broker:
  host: str
  port: int
  tls: bool
  username: Optional[str]
  password: Optional[str]
mqtt:
  topic: str
  qos: int
payload:
  dcu_id: str
  meter_id_source: str (path to CSV)
```
